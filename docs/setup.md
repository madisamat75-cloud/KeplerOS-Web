# System Setup Guide

## Core Philosophy
KeplerOS is built on a "minimalist-first" approach. We strip away all non-essential packages to reduce the attack surface and improve resource allocation.

## Initial Configuration
1. **Kernel:** Using generic kernel with specific compiler optimizations.
2. **GPU:** NVIDIA proprietary drivers configured for KDE Plasma.
3. **Bloatware:** System is currently undergoing aggressive trimming (targeting < 1000 packages).
